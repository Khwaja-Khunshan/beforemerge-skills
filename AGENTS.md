# BeforeMerge

An open-source, AI-native code review knowledge base.

## Available Skills

- **nextjs-review**: Code review rules for Next.js, React, and TypeScript. Install with `npx skills add BeforeMerge/beforemerge --skill nextjs-review`

## Structure

Each skill in `skills/` contains:
- `SKILL.md` — Agent metadata and trigger description
- `AGENTS.md` — Compiled rules document (generated by `node scripts/build.js`)
- `rules/` — Individual rule files organized by category
- `metadata.json` — Version and organization info

## Building

```bash
# Build all skills
node scripts/build.js

# Build a specific skill
node scripts/build.js nextjs-review
```
